Tecnologias

Back-end: Laravel

Laravel é um framework PHP de código aberto, criado por Taylor Otwell, lançado em 2011. É amplamente utilizado no desenvolvimento de aplicações web e segue o padrão MVC (Model-View-Controller).

Linguagens usadas: PHP

  • PHP
Site oficial do Laravel

Back-end: ASP.NET

ASP.NET é um framework da Microsoft para o desenvolvimento de aplicações web, lançado em 2002. Oferece suporte a várias linguagens de programação, incluindo C# e Visual Basic.

Linguagens usadas: C#, Visual Basic

  • C#
  • Visual Basic
Site oficial do ASP.NET

Back-end: Spring MVC

Spring MVC é um framework Java para o desenvolvimento de aplicações web, lançado em 2005. Ele segue o padrão de arquitetura MVC (Model-View-Controller) e é amplamente utilizado na construção de aplicativos empresariais escaláveis e robustos.

Linguagens usadas: Java

  • Java
Site oficial do Spring MVC

Front-end: Angular

Angular é um framework JavaScript de código aberto mantido pelo Google, lançado em 2010. É amplamente utilizado no desenvolvimento de aplicações web de uma única página (SPA - Single Page Applications).

Linguagens usadas: TypeScript

  • TypeScript
Site oficial do Angular

Front-end: React

React é uma biblioteca JavaScript de código aberto, mantida pelo Facebook, lançada em 2013. É amplamente utilizada no desenvolvimento de interfaces de usuário interativas e reativas.

Linguagens usadas: JavaScript, TypeScript

  • JavaScript
  • TypeScript
Site oficial do React

Front-end: Vue

Vue é um framework JavaScript de código aberto, lançado em 2014. É conhecido por sua simplicidade e flexibilidade, permitindo o desenvolvimento de interfaces de usuário interativas e componentizadas.

Linguagens usadas: JavaScript

  • JavaScript
Site oficial do Vue

Front-end: Svelte

Svelte é um framework JavaScript de código aberto, lançado em 2016. Ele se diferencia por compilar os componentes em tempo de construção, gerando um código otimizado para a execução.

Linguagens usadas: JavaScript

  • JavaScript
Site oficial do Svelte

Outras: WordPress

WordPress é uma plataforma de gerenciamento de conteúdo (CMS - Content Management System) de código aberto, lançada em 2003. É amplamente utilizado na criação de blogs, sites e lojas virtuais.

Linguagens usadas: PHP

  • PHP
Site oficial do WordPress

Outras: OutSystems

OutSystems é uma plataforma de desenvolvimento low-code, lançada em 2001. Ela permite o desenvolvimento rápido de aplicações web e móveis, com pouca necessidade de codificação manual.

Linguagens usadas: Visual Basic, C#

  • Visual Basic
  • C#
Site oficial do OutSystems

Outras: Weebly

Weebly é uma plataforma de criação de sites e lojas virtuais, lançada em 2006. Ela oferece uma interface intuitiva e ferramentas simples para criar sites profissionais sem a necessidade de conhecimentos técnicos avançados.

Linguagens usadas: HTML, CSS, JavaScript

  • HTML
  • CSS
  • JavaScript
Site oficial do Weebly

Outras: Wix

Wix é uma plataforma de criação de sites, lançada em 2006. Ela permite criar sites personalizados usando um construtor visual intuitivo, além de oferecer recursos de hospedagem e domínio.

Linguagens usadas: HTML, CSS, JavaScript

  • HTML
  • CSS
  • JavaScript
Site oficial do Wix
